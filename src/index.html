Figura Addons

Figura Badge Backport Tool

The first of the backport collection, porting the /figura badge command from 0.1.6. Also my first venture into wrangling lots of platforms and Minecraft versions in a single JAR.

Figura v5 support

Implements support for Blockbench model files created on Blockbench version 5.0.0 or later. (Figura fails to load these files by default.) Utilized

fsb-carpet-addon

A tiny FSB plugin for (upcoming) 0.1.6 that makes it easier to use fake players with FSB.

Figura Development (PRs)

Features

Blockbench project file parser rework (5.0 support)

The code that drives development of the aformentioned addon mod. Teaches Figura to load the new model format, and adds a few QOL changes along the way, like much better keyframe errors.

Expanded Texture APIs

Lets users do cool things with textures that were annoying and expensive before. Pallete swaps, scaling algorithms, mathematical operations, patterns, line drawing, and the powerful multitool :blit().

Deny uploading errored avatars by default

One of a few QOL changes to make it easier for new users to avoid tripping over themselves and help them solve issues before publishing their avatar to everyone else.

Performance & Fixes

Lua in Animations Optimization

Made animations containing Lua drivers for translation, rotation, and scale much more performant, after a bug report in the Figura discord.
